# APK Build Information - Tilt Maze

## Build Status

Die APK wurde mit Expo Application Services (EAS) erstellt.

## Build-Befehle

Verwendete Build-Befehle (siehe auch docs/04_build_apk_eas.md):

```bash
# EAS CLI installieren (einmalig)
npm install -g eas-cli

# Bei Expo anmelden
eas login

# APK erstellen (Preview-Profil)
eas build --platform android --profile preview
```

## Build-Konfiguration

Das Preview-Profil in `eas.json` ist wie folgt konfiguriert:

```json
{
  "build": {
    "preview": {
      "distribution": "internal",
      "android": {
        "buildType": "apk"
      }
    }
  }
}
```

## APK-Download

Nach erfolgreichem Build wird die APK über die Expo-Website verfügbar:

**Download-Link:** https://expo.dev/accounts/[username]/projects/app-m335/builds

**Alternative:** 
- Öffne https://expo.dev und navigiere zu deinem Projekt
- Wähle "Builds" im Menü
- Suche nach dem neuesten Android Preview Build
- Klicke auf "Download" um die APK herunterzuladen

## Installation

1. APK-Datei von Expo herunterladen
2. Datei auf Android-Gerät übertragen (USB, Cloud, etc.)
3. In Einstellungen "Installation aus unbekannten Quellen" aktivieren
4. APK-Datei öffnen und "Installieren" wählen
5. App starten und testen

## Hinweise

- Die APK wird auf Expo-Servern für 30 Tage gespeichert
- Für langfristige Speicherung: APK lokal herunterladen und archivieren
- Für Play Store Veröffentlichung: Production-Profil mit AAB-Format verwenden
